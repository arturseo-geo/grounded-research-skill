# Security Policy

## Scope

This is a Claude Code skill — a collection of markdown instruction files.
It contains no executable code, no authentication tokens, and no user data.

Security concerns for this repo are limited to:
- Instructions that could mislead users into unsafe research decisions
- Constraint logic that could be bypassed to produce unreliable output

## Reporting an Issue

If you find content in this skill that could cause harm — a flawed constraint,
a misleading instruction, or anything that undermines the accuracy guarantees —
please report it privately before opening a public issue.

**Contact**: Open a private security advisory via
[GitHub Security Advisories](https://github.com/arturseo-geo/grounded-research-skill/security/advisories/new)
or email **artur@thegeolab.net**, or reach out via [𝕏 @TheGEO_Lab](https://x.com/TheGEO_Lab).

We aim to address valid reports within 48 hours.
